


### Wget
```bash
wget <url>

```
### Wget Magic
wget -r --convert-links <url>

### Wget Output file

```bash
wget <url> -O outputfilename.ext
```

### Wget adjust extension

```bash
wget --adjust-extension <url> 
```

### Youtube-dl

```bash
 $ youtube-dl <url>

```

#### List Download format

```bash
 $ youtube-dl -F <url>
```

#### Download in a specific format
```bash
 $ youtube-dl -f <format-code> <url>
```



### SoundCloud-Dl

```bash
 $ scdl -l <url>

```
