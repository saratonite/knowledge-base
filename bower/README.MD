# Bower

### Install bower 
```bash
npm install -g bower
```


### Initialize bower.json
```bash
bower init
```

### Intsall bower package
```bash
bower install <packagename>
```
### Intsall bower package and update bower.json
```bash
bower install <packagename> --save
```
